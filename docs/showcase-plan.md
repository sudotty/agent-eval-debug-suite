# Showcase Plan

## Product form

A small console for comparing two runs of the same AI workflow.

## MVP screens

1. Case list.
2. Run comparison.
3. Step timeline.
4. Result table.
5. Summary note.

## Demo flow

1. Load sample cases.
2. Run the first version.
3. Run the second version.
4. Compare the result table.
5. Open one case and inspect the timeline.
6. Show the summary note.

## Good demo points

- Work is checked with repeatable cases.
- Steps are easy to inspect.
- The result is easy to explain.
